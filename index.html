  • The machine learning for bioinformatics course project [my code]
    • This project was about drug-target binding prediction. One of the methods used in solving this problem is the DeepDTA model. which extracts features from drug and target molecular structures using a two-branch CNN network. However, this model doesn't take the sequential nature of target amino-acid structures into account. In this project, in addition to implementing some of the previous methods in the drug-target binding problem, I designed a model which uses an RNN network to extract sequential features of amino-acids. Eventually, These features, are augmented with the features extracted from the two-branch CNN network and used to predict the binding value of a drug-target pair.
